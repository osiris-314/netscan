# COMPREHENSIVE NETWORK SCANNER
To scan the whole network
```
python netscan.py
```
To only scan specific ip
```
python netscan.py xx.xx.xx.xx
```
To scan all ips in a txt file
```
python netscan.py -f ip_file_here.txt
```
